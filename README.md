## 🛠️ Key Architectural Features

* **3D Particle Universe:** Interactive Three.js particle array reactive to mouse movements and dynamic window scrolling behavior.
* **Neural Core Network:** Pure WebGL wireframe constellation representing dynamic skill node ecosystems.
* **Glassmorphic UI Design:** High-end frosted glass layout system utilizing hardware-accelerated backdrop filters (`backdrop-filter: blur`).
* **Automated Data Logger:** Submission engine inside the Contact Hub that automatically generates, encrypts, and downloads message logs directly into an Excel-compatible `.csv` payload file.
* **Micro-Interactions:** Smooth page transitions driven entirely by **GSAP (GreenSock)** and hardware-optimized custom 3D card hover-tilt mechanics.
